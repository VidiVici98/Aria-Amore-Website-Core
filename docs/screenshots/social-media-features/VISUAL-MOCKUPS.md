## Interaction States

**Social Platform Cards:**
1. Default: White background, light gray border, subtle shadow
2. Hover: Lifts 8px, gold border, icon rotates 5Â°, button scales 1.05x
3. Click: Opens link in new tab

**Share Buttons:**
1. Default: White background, icon + label
2. Hover: Platform-specific gradient, white text, scale 1.05x
3. Click: Opens share dialog or copies link

**Chat Button:**
1. Default: Gold gradient, pulse shadow animation
2. Hover: Lifts 4px, deeper shadow
3. Click: Opens contact modal

**Contact Modal:**
1. Appear: Fade in background, slide up content
2. Options Default: White with gray border
3. Options Hover: Gold border, slide right 5px, arrow appears
4. Close: Click outside, ESC key, or close button (X)

## Responsive Behavior

**Breakpoints:**
- Desktop (1024px+): 2-column cards, 4-column share buttons, full chat text
- Tablet (768-1023px): 2-column cards, 3-column share buttons
- Mobile (<768px): 1-column cards, 3-column share buttons, icon-only chat

**Touch Targets:**
- All interactive elements: Minimum 48x48px (WCAG AAA)
- Share buttons mobile: Minimum 80px height
- Chat button mobile: 56x56px
- Modal options: Full-width, 60px+ height
