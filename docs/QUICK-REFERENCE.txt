TROUBLESHOOTING
Problem: Animation doesn't play
  → Hard refresh browser (Ctrl+Shift+R)
  → Check browser console for errors (F12)

Problem: Pages show 404 errors  
  → Make sure npm start is running
  → Check serving from public/ folder

Problem: npm start fails
  → Run: npm install
  → Run: npm list http-server

Problem: Assets not loading
  → Check symlinks: ls -la public/assets
  → Verify CSS file exists: public/assets/css/styles.css
